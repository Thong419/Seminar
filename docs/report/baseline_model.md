# Baseline Model

The baseline model uses TF-IDF features with Logistic Regression. It provides a strong, interpretable reference point for the transformer model and helps validate that the dataset and pipeline are functioning correctly.

## Why It Matters

- Fast to train and evaluate.
- Easy to interpret.
- Provides a meaningful comparison against RoBERTa.

## What to Report

- TF-IDF vectorizer configuration.
- Logistic Regression hyperparameters.
- Validation and test performance metrics.
