# Related Work

Prior fake news research commonly uses classical machine learning models, transformer encoders, evidence-based verification systems, and explainability methods. Traditional approaches such as TF-IDF with linear classifiers are efficient and interpretable, but they may underperform on nuanced language. Transformer models such as RoBERTa improve contextual understanding and usually yield stronger classification performance.

Recent systems increasingly combine classification with retrieval and explanation so that the output can be reviewed by humans. This project follows that direction by integrating agentic orchestration, evidence retrieval, token-level explanation, monitoring, and responsible AI safeguards into a single system.
