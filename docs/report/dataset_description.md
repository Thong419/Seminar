# Dataset Description

The project uses a labeled fake news dataset with two classes: fake and real. The dataset is treated as a binary classification problem and is split into training, validation, and test sets.

## Dataset Properties

- Input: article text.
- Output labels: fake, real.
- Preprocessing: lowercasing, URL removal, HTML cleaning, whitespace normalization, and special-character handling.

## Notes for Reporting

- Report the exact dataset version used in the experiments.
- State the number of samples per class if available in the final experiment logs.
- Mention any class imbalance and how it was handled.
