# Demo Checklist

## Startup Sequence

1. Start backend and frontend.
2. Confirm backend health.
3. Open Streamlit UI.
4. Load a sample article.
5. Run an analysis.

## Sample Fake News

- Sensational miracle cure claim with no sources.

## Sample Real News

- Article with explicit references to verified reporting and official statements.

## Expected Outputs

- Prediction badge.
- Confidence score.
- Trust score.
- Important tokens.
- Evidence cards.
- Final explanation.

## Fallback Plan if Internet Retrieval Fails

- Show the prediction and explanation with limited evidence.
- Explain that retrieval was unavailable and continue the demo with monitoring and explainability.
